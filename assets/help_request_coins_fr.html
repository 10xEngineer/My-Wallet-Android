En utilisant cette boîte de dialogue, vous pouvez recevoir des bitcoins (vous devez avoir installé Bitcoin Wallet).

D'abord, saisissez le montant de Bitcoins à recevoir.

Ensuite, l'autre personne doit scanner le QR code généré. Ou bien, vous pouvez envoyer une URL Bitcoin à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL Bitcoin.

Dasn les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des bitcoins avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer la transaction.
