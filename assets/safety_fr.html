Quelques mots sur la sécurité de vos Bitcoins:

  • Bitcoins dans le Bitcoin Wallet sont stockés dans le téléphone. Si vous le perdez, vous perdez vos Bitcoins. Comme de l'argent liquide dans votre porte-feuille.
  • Avant de désinstaller Bitcoin Wallet (ou d'effacer les données de l'application), transferez vos Bitcoins dans un autre wallet. Tout bitcoins restant seront perdus.
  • Avant de mettre à jour Bitcoin Wallet, consultez toujours les changements récents sur Android Market! N'activez pas la mise à jour automatique.
  • Les transactions sont irréversibles. Si vous envoyez vos Bitcoins dans le vide,il n'y a aucun moyen de les récupérer sauf si vous connaissez le destinataire et qu'il est d'accord pour les renvoyer.
  • Gardez votre téléphone Android en sécurité! N'installez que des applications de confiance. Des applications pirates pourraient essayer de voler votre wallet.
  • Minimisez les risques! Utilisez Android Wallet seulement pour des petits montants pour un usage quotidien, et gardez vos économies en sûreté.
  • Si vous voulez seulement essayer Bitcoin sans risque (mais sans la possibilité de payer pour de vrai), vous pouvez essayer Bitcoin Wallet pour Testnet (application distincte).
