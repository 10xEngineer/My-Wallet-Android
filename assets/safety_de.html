Hinweise zur Sicherheit:

  • Bitcoins in Bitcoin Wallet werden lokal auf dem Gerät gespeichert. Wenn das Gerät verloren geht, gehen alle Bitcoins verloren. So wie bei einem traditionellen Portmonnaie.
  • Vor der Deinstallation von Bitcoin Wallet (oder Löschen der App-Daten/Formatierung des Geräts), müssen alle Bitcoins auf ein anderes Wallet übertragen werden. Auf dem Gerät verbleibende Bitcoins gehen verloren.
  • Vor der Aktualisierung von Bitcoin Wallet im Android Market sollten die Neuigkeiten beachtet werden. Von automatischen Updates wird abgeraten.
  • Transaktionen können nicht rückgängig gemacht werden. Wenn Bitcoins ins Unbekannte geschickt werden, gibt es keine Möglichkeit, sie zurückzuholen, es sei denn, der Empfänger ist bekannt und erklärt sich zur Rücksendung bereit.
  • Das verwendete Android-Gerät muss sicher sein! Es sollten nur Apps installiert werden, denen vollkommen vertraut wird. Bösartige Apps könnten versuchen, die in der Wallet gespeicherten Bitcoins zu stehlen.
  • Das Verlustrisiko kann minimiert werden, indem nur geringe Beträge für den täglichen Gebrauch auf Bitcoin Wallet gespeichert werden. Größere Beträge sollten an einem sichererem Ort aufbewahrt werden.
  • Um Bitcoin ohne Risiko auszuprobieren, kann Bitcoin Wallet für Testnet als separate App heruntergeladen werden. Echte Zahlungen sind damit jedoch nicht möglich.
